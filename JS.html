JS (Java script)

Java Script é a linguagem responsavel por dar comportamento interatividade ás paginas web, permitindo responder a ação do usuário (como cliques e digitação), manipular o conteudo do HTML em tempo real e se comunicar com servidores;

Em resumo, enquanto HTML estutura e CSS estiliza, o JavaScript faz a página "funcionar"
