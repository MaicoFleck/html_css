JS

JavaScript é a linguagem responsável por dar comportamento e interatividade às páginas web, permitindo responder a ações do usuário, manipular o conteúdo em tempo real e comunicar com servidores.

Enquanto o HTML estrutura e o CSS estiliza, o JavaScript é o motor que faz a página funcionar.
